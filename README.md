# beyzasimsekoglu.github.io

Personal site for H. Beyza Simsekoglu — product manager, Cornell Tech '26.

Static single-page site. Just `index.html`. No build step.

## Local preview

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## Deploy

This repo is named `beyzasimsekoglu.github.io`, so GitHub Pages serves it automatically at <https://beyzasimsekoglu.github.io> from the `main` branch root. Push and wait ~60 seconds.

## Edit

Everything lives in `index.html` — content, styles, scripts. Search for the section you want and edit in place.
